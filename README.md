# Trillion Games 2D - 通用2D游戏引擎脚手架

一个基于 LÖVE2D 的通用2D游戏开发脚手架，第一个落地产品是中国象棋。

## 项目特点

- **分层架构**：通用层与业务层严格分离，新游戏快速复用
- **中文注释**：对Lua初学者友好，每个特性都有JS/Java/Python类比说明
- **增量开发**：每个版本都有明确可运行的成果
- **多端发布**：支持Steam/桌面/Android

## 快速开始

### 环境要求
- LÖVE2D 11.5+（下载地址：https://love2d.org/）

### 运行游戏
```bash
# 在项目根目录执行
love .
```

### 操作说明
- **按 1**：切换到游戏场景
- **按 2**：切换到菜单场景
- **ESC**：返回菜单 / 退出游戏
- **F2**：截图
- **鼠标点击**：游戏场景里点击画点（测试输入用）
- **上下键/回车**：菜单选择

## 项目结构

```
love2DGame_wren/
├── docs/              # 项目文档
├── src/               # 源代码
│   ├── core/          # 核心层（通用）
│   ├── systems/       # 系统层（通用）
│   ├── scenes/        # 场景层
│   ├── entities/      # 实体层（业务）
│   └── game/          # 游戏逻辑层（业务）
├── assets/            # 资源文件（图片/音效/字体）
├── tests/             # 测试
├── conf.lua           # LÖVE2D 配置
└── main.lua           # 游戏入口
```
